   # Detection of Damaged Buildings in Al-Shati Camp

## Project Description
This project aims to detect and assess damaged buildings in Al-Shati Refugee Camp (Gaza) using satellite imagery and AI techniques.

## Objectives
- Extract building footprints from pre-event images
- Detect damaged areas from post-event images
- Identify damaged buildings using overlay analysis
- Estimate reconstruction costs

## Tools
- QGIS
- Satellite Images
- OpenStreetMap
- AI Models
